# MeasurementTools

Counting is difficult. This is a Minecraft Fabric mod to make measuring and counting easier.

## Features

- **Block Selection**: Select multiple blocks to create measurement regions
- **Shape Modes**: Rectangle, Cylinder, and Ellipsoid wireframe displays
- **Distance Labels**: Shows measurements on each edge
- **Subdivisions**: Divide regions into equal parts with visual guides
- **Radial Menu**: Hold the measure key to access options wheel

## Controls

Hotkeys are configurable in the in-game menu.

- **G (tap)**: Select the block you're looking at
- **G (hold)**: Open radial options menu
- **H**: Clear selection

## Shape Modes

- **Rectangle**: Bounding box around all selected blocks
- **Cylinder**: First block = center, furthest block = radius
- **Ellipsoid**: 3D ellipsoid using selection bounds

## Requirements

- Minecraft 1.21.10
- Fabric Loader 0.18.4+
- Fabric API
